# Java 线程的 6 种状态及切换

这6种状态定义在 Thread 类的 State 枚举中

```java
    public enum State {
        
        NEW,

        RUNNABLE,

        BLOCKED,

        WAITING,

        TIMED_WAITING,

        TERMINATED;
    }
```
### 状态说明

#### 1. 初始状态
实现 Runnable 接口和继承 Thread 可以得到一个线程类，new 一个实例出来，线程就进入了初始状态。
